# E-Commerce Business Insights Report

## Project Overview
This project was completed as part of a Data Analytics Internship Program. The objective of this project is to analyze Global E-Commerce Orders data and generate meaningful business insights using SQL and Power BI.

## Dataset Information
- Dataset: Global E-Commerce Orders (2011–2014)
- Records: 51,271+ Orders
- Domain: E-Commerce

## Project Objectives
- Analyze sales and profit performance
- Identify top-performing products and customers
- Study regional and market-wise sales trends
- Evaluate discount impact on profitability
- Generate actionable business insights

## Tools & Technologies Used
- Microsoft Excel
- SQL (MySQL)
- Power BI

## Project Workflow

### 1. Data Cleaning
- Removed duplicate records
- Checked missing values
- Standardized data format
- Prepared clean dataset for analysis

### 2. SQL Analysis
Performed analysis on:
- Top 10 Profitable Products
- Top 10 Customers by Sales
- Region-wise Total Sales
- Category-wise Average Profit
- Highest Discount Category
- Orders with Negative Profit
- Monthly Sales Trend
- Market-wise Revenue Analysis
- Top Performing Sub-Categories
- Ship Mode Usage Analysis

### 3. Power BI Dashboard
Created interactive dashboards for:
- Sales Analysis
- Profit Analysis
- Customer Insights
- Market Performance
- Product Performance

### 4. Business Insights
Generated business recommendations based on data findings and trends.

## Key Outcomes
- Identified high-profit products and categories
- Analyzed customer purchasing behavior
- Evaluated regional and market performance
- Tracked monthly sales trends
- Highlighted areas for business improvement
